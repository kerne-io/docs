<Note>
The React SDK does not currently expose a usage tracking method. Use the Server SDK to retrieve usage data.
</Note>

For usage tracking, use the [Server SDK billing.usage()](/sdks/server/billing/consumption/usage) method in your backend API routes.

## Server-side example

```typescript
// In your API route
import { kerne } from '@/lib/kerne'; // Server SDK

export async function GET(req: Request) {
  const usage = await kerne.billing.usage('api_calls');
  return Response.json(usage);
}
```

## Client-side consumption

To display usage in your React app, fetch it from your API:

```tsx
import { useState, useEffect } from 'react';

function UsageDisplay() {
  const [usage, setUsage] = useState(null);

  useEffect(() => {
    fetch('/api/usage')
      .then(res => res.json())
      .then(setUsage);
  }, []);

  if (!usage) return <p>Loading...</p>;

  return (
    <div>
      <h3>API Usage</h3>
      <p>{usage.used} / {usage.limit ?? 'âˆž'} calls</p>
    </div>
  );
}
```
