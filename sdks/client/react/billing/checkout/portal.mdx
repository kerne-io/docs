---
title: "Portal"
description: "Manage subscriptions via Stripe"
---

The `createPortal` method generates a link to the Stripe Customer Portal, where users can manage their payment methods, invoices, and active subscriptions.

## Signature

```typescript
kerne.billing.createPortal(options?: PortalOptions): Promise<PortalResponse>
```

## Parameters

| Name | Type | Description |
|------|------|-------------|
| `returnUrl` | `string` | The URL to return to after the user exits the portal. Defaults to the current URL. |

## Response

Returns a `Promise` that resolves to a `PortalResponse`.

```typescript
interface PortalResponse {
  portalUrl: string; // The URL to redirect the user to
}
```

## Possible errors

| Code | Status | Description |
|------|--------|-------------|
| `K7001` | 502 | **Stripe error**: An error occurred while communicating with Stripe. |
| `K4001` | 404 | **Resource not found**: The user does not have an active or past subscription. |
| `K3004` | 403 | **Operation not allowed**: The customer portal is not enabled for this application. |

## Example

```typescript
const { portalUrl } = await kerne.billing.createPortal({
  returnUrl: `${window.location.origin}/settings/billing`
});

// Redirect the user to the Stripe-hosted portal
window.location.href = portalUrl;
```
