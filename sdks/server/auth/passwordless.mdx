---
title: "Passwordless"
description: "Implement magic link authentication"
---

Passwordless authentication allows users to log in with a magic link sent to their email.

## Start flow

Initiate the flow by sending a magic link email.

```typescript
await kerne.auth.startPasswordless('user@example.com');
```

## Verify token

Verify the token when the user clicks the link.

```typescript
const { user, token } = await kerne.auth.verifyPasswordless('token_from_url');
```
