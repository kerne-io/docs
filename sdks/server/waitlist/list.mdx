## Request

<CodeGroup>
```typescript Node.js
const result = await kerne.waitlist.list({
  limit: 20,
  status: 'PENDING'
});
```
</CodeGroup>

## Response

Returns a list of `WaitlistEntry` objects and pagination metadata.
