---

## Request

<CodeGroup>
```typescript Node.js
const entry = await kerne.waitlist.join({
  email: 'alice@company.com',
  name: 'Alice Doe'
});
```
</CodeGroup>

## Response

Returns a `WaitlistEntry` object.

<ResponseField name="id" type="string" required>
  Unique ID of the entry.
</ResponseField>

<ResponseField name="status" type="string" required>
  `PENDING`, `INVITED`, or `REGISTERED`.
</ResponseField>
