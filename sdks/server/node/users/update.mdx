---
title: "Update user"
description: "Modify a user profile"
---

The `update` method allows you to modify an existing user's profile or status.

## Signature

```typescript
kerne.users.update(id: string, params: UpdateUserParams): Promise<User>
```

## Parameters

| Name | Type | Description |
|------|------|-------------|
| `name` | `string` | The user's full name. |
| `role` | `string` | Update the user's role. |
| `isActive` | `boolean` | Set to `false` to deactivate/ban the user. |

## Response

Returns a `Promise` that resolves to the updated `User` object.

## Example

```typescript
// Deactivate a user
await kerne.users.update('usr_123', {
  isActive: false
});

// Promote a user to Admin
await kerne.users.update('usr_456', {
  role: 'ADMIN'
});
```
