title: "Waitlist"
description: "Manage waitlist entries for restricted registration"
---

The `waitlist` module allows you to manage waitlist entries when registration is restricted. Users can join the waitlist publicly, while administrators can list and remove entries.

For more information on registration modes, see the [Registration modes](/concepts/identity/registration-modes) concept page.

## Join waitlist

Add an email to the waitlist. This is a public method that doesn't require authentication.

### Signature

```typescript
kerne.waitlist.join(params: JoinWaitlistParams): Promise<WaitlistEntry>
```

### Parameters

| Name | Type | Required | Description |
|------|------|----------|-------------|
| `email` | `string` | Yes | The email address to add to the waitlist. |
| `name` | `string` | No | Optional name for the waitlist entry. |
| `metadata` | `object` | No | Optional metadata to attach to the entry. |

### Example

```typescript
const entry = await kerne.waitlist.join({
  email: 'user@example.com',
  name: 'John Doe',
});

console.log(`Added to waitlist with status: ${entry.status}`);
```

## List waitlist entries

Retrieve all waitlist entries. Requires secret key authentication.

### Signature

```typescript
kerne.waitlist.list(params?: ListWaitlistParams): Promise<WaitlistListResponse>
```

### Parameters

| Name | Type | Required | Description |
|------|------|----------|-------------|
| `limit` | `number` | No | Maximum entries to return (default: 50). |
| `offset` | `number` | No | Number of entries to skip for pagination. |
| `status` | `string` | No | Filter by status: `PENDING`, `INVITED`, or `REGISTERED`. |

### Example

```typescript
const { entries, pagination } = await kerne.waitlist.list({
  status: 'PENDING',
  limit: 20,
});

entries.forEach(entry => {
  console.log(`${entry.email} - ${entry.status}`);
});
```

---

## Remove from waitlist

Remove a waitlist entry by ID. Requires secret key authentication.

### Signature

```typescript
kerne.waitlist.remove(id: string): Promise<void>
```

### Example

```typescript
await kerne.waitlist.remove('wl_01234567890abcdef');
```

---

## Possible errors

| Code | Status | Description |
|------|--------|-------------|
| `K2005` | 409 | **Duplicate entry**: Email is already on the waitlist. |
| `K1008` | 409 | **Already registered**: Email is already registered as a user. |
| `K4005` | 404 | **Entry not found**: The waitlist entry ID does not exist. |
| `K3002` | 403 | **Insufficient permissions**: Secret key required for admin operations. |
