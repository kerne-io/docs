---
title: "Invitations"
description: "Manage invitation tokens and codes for controlled registration"
---

The `invitations` module allows you to create and manage invitation tokens and codes for controlled user registration. Validation methods are public, while creation and management require secret key authentication.

For more information on registration modes, see the [Registration modes](/concepts/identity/registration-modes) concept page.

## Validate token

Check if an invitation token is valid. This is a public method.

### Signature

```typescript
kerne.invitations.validateToken(token: string): Promise<ValidateTokenResponse>
```

### Response

```typescript
interface ValidateTokenResponse {
  valid: boolean;
  email?: string;  // Present if valid
}
```

### Example

```typescript
const result = await kerne.invitations.validateToken('eyJhbGciOiJIUzI1NiI...');

if (result.valid) {
  console.log(`Token is valid for: ${result.email}`);
}
```

## Validate code

Check if an invitation code is valid and can be used. This is a public method.

### Signature

```typescript
kerne.invitations.validateCode(code: string): Promise<ValidateCodeResponse>
```

### Response

```typescript
interface ValidateCodeResponse {
  valid: boolean;
  canUse: boolean;  // false if already used
}
```

### Example

```typescript
const result = await kerne.invitations.validateCode('ABC12345');

if (result.valid && result.canUse) {
  console.log('Code is valid and can be used');
}
```

## Create token

Create an invitation token for a specific email. Requires secret key authentication.

### Signature

```typescript
kerne.invitations.createToken(params: CreateTokenParams): Promise<{
  token: string;
  expiresAt: string;
  email: string;
}>
```

### Parameters

| Name | Type | Required | Description |
|------|------|----------|-------------|
| `email` | `string` | Yes | The email address to invite. |

### Example

```typescript
const invitation = await kerne.invitations.createToken({
  email: 'newuser@example.com',
});

console.log(`Invitation sent! Expires: ${invitation.expiresAt}`);
```

---

## Create code

Create a single-use invitation code. Requires secret key authentication.

### Signature

```typescript
kerne.invitations.createCode(params?: CreateCodeParams): Promise<{
  code: string;
  expiresAt: string | null;
}>
```

### Parameters

| Name | Type | Required | Description |
|------|------|----------|-------------|
| `expiresAt` | `string` | No | ISO date string for code expiration. |

### Example

```typescript
const { code } = await kerne.invitations.createCode({
  expiresAt: '2025-01-15T00:00:00.000Z',
});

console.log(`Share this code: ${code}`);
```

## List invitations

List all invitation tokens and codes. Requires secret key authentication.

### Signature

```typescript
kerne.invitations.list(): Promise<InvitationsListResponse>
```

### Response

```typescript
interface InvitationsListResponse {
  tokens: InvitationToken[];
  codes: InvitationCode[];
}
```

### Example

```typescript
const { tokens, codes } = await kerne.invitations.list();

console.log(`${tokens.length} tokens, ${codes.length} codes`);
```

---

## Revoke invitation

Revoke an invitation by ID. Requires secret key authentication.

### Signature

```typescript
kerne.invitations.revoke(id: string): Promise<void>
```

### Example

```typescript
await kerne.invitations.revoke('inv_01234567890abcdef');
```

## Possible errors

| Code | Status | Description |
|------|--------|-------------|
| `K1008` | 409 | **Already registered**: Email is already registered. |
| `K4004` | 404 | **Invitation not found**: The invitation ID does not exist. |
| `K3002` | 403 | **Insufficient permissions**: Secret key required for admin operations. |
