Prices define the billing amount, currency, and interval for a plan. Plans can have multiple prices (e.g., monthly and annual options).

<Info>
Plan prices can be fully managed through the Kerne dashboard. Unless you're building a custom admin interface, you typically won't need to use these endpoints directly.
</Info>

## List prices

Retrieve all prices, optionally filtered by plan.

<CodeGroup>
```typescript Node.js
// List all prices
const allPrices = await kerne.billing.prices.list();

// List prices for a specific plan
const planPrices = await kerne.billing.prices.list('plan_123');
```
</CodeGroup>

**Response**

```typescript
interface PlanPrice {
  id: string;
  plan_id: string;
  amount: number;
  currency: string;
  interval: 'month' | 'year' | 'week' | 'day';
  is_active: boolean;
  trial_days: number | null;
  provider_price_id: string | null;
  created_at: string;
  updated_at: string;
}
```

```json
[
  {
    "id": "price_123",
    "plan_id": "plan_abc",
    "amount": 2900,
    "currency": "usd",
    "interval": "month",
    "is_active": true,
    "trial_days": 14,
    "provider_price_id": "price_stripe_123",
    "created_at": "2025-01-01T00:00:00.000Z",
    "updated_at": "2025-01-01T00:00:00.000Z"
  }
]
```

## Create price

Create a new price for a plan.

<CodeGroup>
```typescript Node.js
const response = await kerne.billing.prices.create({
  plan_id: 'plan_123',
  amount: 2900,      // $29.00
  currency: 'usd',
  interval: 'month',
  trial_days: 14
});
const price = response.data;
```
</CodeGroup>

**Parameters**

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `plan_id` | string | Yes | Parent plan ID |
| `amount` | number | Yes | Price in cents (e.g., 2900 = $29.00) |
| `currency` | string | Yes | ISO currency code (`usd`, `eur`, etc.) |
| `interval` | enum | Yes | `month`, `year`, `week`, `day` |
| `trial_days` | number | No | Override plan's trial period |
| `provider_price_id` | string | No | Stripe/provider price ID |

**Response**: Returns the created `PlanPrice` object.
