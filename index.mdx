## What is Kerne?

Kerne provides the infrastructure you need to build and scale monetized software. It unifies authentication and billing into a single, developer-first platform.

Instead of stitching together disparate auth and billing providers, Kerne gives you a unified data model. Your users and their subscriptions are deeply integrated from day one.

## SDK example

<CodeGroup>
```typescript Server (@kerne/server)
import { Kerne } from '@kerne/server';

const kerne = new Kerne({
  appId: process.env.KERNE_APP_ID,
  secretKey: process.env.KERNE_SECRET_KEY
});

// Create a user and grant access
const user = await kerne.users.create({
  email: 'user@example.com',
  password: 'SecurePassword123'
});
```

```typescript Client (@kerne/react)
import { KerneProvider, useAuth } from '@kerne/react';

// 1. Wrap your application
function Root() {
  return (
    <KerneProvider appId={process.env.NEXT_PUBLIC_KERNE_APP_ID}>
      <App />
    </KerneProvider>
  );
}

// 2. Use the hook in your components
function Profile() {
  const { user, login } = useAuth();
  
  if (!user) return <button onClick={login}>Login</button>;
  return <div>Welcome, {user.email}</div>;
}
```
