---
title: "List Active Plans"
api: "GET /v1/pricing/plans/{productId}"
description: "List all active plans with prices and entitlements for a specific product"
---

## Path Parameters

<ParamField path="productId" type="string" required>
  The ID of the product to list plans for
</ParamField>

## Response

<ResponseField name="plans" type="PublicPlan[]">
  A list of active plans with restricted metadata suitable for public display.
</ResponseField>

### PublicPlan

<ResponseField name="id" type="string">
  The plan ID
</ResponseField>

<ResponseField name="slug" type="string">
  The plan slug (e.g. "free", "pro")
</ResponseField>

<ResponseField name="name" type="string">
  The plan display name
</ResponseField>

<ResponseField name="description" type="string">
  The plan description
</ResponseField>

<ResponseField name="prices" type="PlanPrice[]">
  Active prices for this plan
</ResponseField>

<ResponseField name="entitlements" type="Entitlement[]">
  Active entitlements for this plan
</ResponseField>

## Errors

| Code | Status | Description |
|------|--------|-------------|
| `K4002` | 404 | **Tenant not found**: The specified tenant slug is invalid. |
