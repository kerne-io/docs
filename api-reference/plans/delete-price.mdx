---
title: "Delete price"
api: "DELETE /v1/admin/billing/plans/{planId}/prices/{priceId}"
---

Delete a price from a plan.

## Authentication

<Note>
This endpoint requires **secret key** authentication (`sk_live_...`).
</Note>

## Path parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `planId` | `string` | Plan ID. |
| `priceId` | `string` | Price ID. |

## Response

```json
{
  "success": true
}
```

## Errors

| Code | Status | Description |
|------|--------|-------------|
| `K4001` | 404 | Plan or price not found. |
| `K3002` | 403 | Insufficient permissions. |
| `K5003` | 400 | Cannot delete price with active subscriptions. |
