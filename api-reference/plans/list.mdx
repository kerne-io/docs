---
title: "List plans"
api: "GET /v1/admin/billing/plans"
---

List all plans in your tenant.

## Authentication

<Note>
This endpoint requires **secret key** authentication (`sk_live_...`).
</Note>

## Response

```json
[
  {
    "id": "plan_01234567890abcdef",
    "product_id": "prod_01234567890abcdef",
    "name": "Pro Monthly",
    "description": "Monthly Pro subscription",
    "is_active": true,
    "features": [
      {
        "feature_id": "feat_abc123",
        "value": true
      }
    ],
    "prices": [
      {
        "id": "price_xyz789",
        "amount": 2900,
        "currency": "usd",
        "interval": "month"
      }
    ],
    "created_at": "2025-01-01T00:00:00.000Z"
  }
]
```

## Errors

| Code | Status | Description |
|------|--------|-------------|
| `K3002` | 403 | Insufficient permissions. |
