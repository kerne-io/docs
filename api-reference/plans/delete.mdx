---
title: "Delete plan"
api: "DELETE /v1/admin/billing/plans/{id}"
---

Delete a plan.

## Authentication

<Note>
This endpoint requires **secret key** authentication (`sk_live_...`).
</Note>

## Path parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `id` | `string` | Plan ID. |

## Response

```json
{
  "success": true
}
```

## Errors

| Code | Status | Description |
|------|--------|-------------|
| `K4001` | 404 | Plan not found. |
| `K3002` | 403 | Insufficient permissions. |
| `K5002` | 400 | Cannot delete plan with active subscriptions. |
