---
title: "Update price"
api: "PUT /v1/admin/billing/plans/{planId}/prices/{priceId}"
---

Update an existing price.

## Authentication

<Note>
This endpoint requires **secret key** authentication (`sk_live_...`).
</Note>

## Path parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `planId` | `string` | Plan ID. |
| `priceId` | `string` | Price ID. |

## Request body

| Parameter | Type | Description |
|-----------|------|-------------|
| `is_active` | `boolean` | Whether the price is active. |

<Warning>
You cannot change the amount or interval of an existing price. Create a new price instead.
</Warning>

## Example request

```json
{
  "is_active": false
}
```

## Response

```json
{
  "id": "price_01234567890abcdef",
  "is_active": false,
  "updated_at": "2025-01-02T00:00:00.000Z"
}
```

## Errors

| Code | Status | Description |
|------|--------|-------------|
| `K4001` | 404 | Plan or price not found. |
| `K2001` | 400 | Invalid request payload. |
| `K3002` | 403 | Insufficient permissions. |
