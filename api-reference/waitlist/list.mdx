---
title: "List waitlist"
api: "GET /v1/waitlist"
---

Retrieve all waitlist entries with optional filtering.

## Authentication

<Note>
This endpoint requires **secret key** authentication (`sk_live_...`).
</Note>

## Query parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `limit` | `integer` | Maximum entries to return (default: 50). |
| `offset` | `integer` | Number of entries to skip for pagination. |
| `status` | `string` | Filter by status: `PENDING`, `INVITED`, or `REGISTERED`. |

## Response

```json
{
  "entries": [
    {
      "id": "wl_01234567890abcdef",
      "email": "interested@example.com",
      "name": "John Doe",
      "status": "PENDING",
      "created_at": "2025-01-01T00:00:00.000Z"
    }
  ],
  "pagination": {
    "total": 1,
    "limit": 50,
    "offset": 0,
    "hasMore": false
  }
}
```

## Errors

| Code | Status | Description |
|------|--------|-------------|
| `K3002` | 403 | Insufficient permissions (requires secret key). |
