---
title: "Sync to Stripe"
api: "POST /v1/admin/billing/products/{id}/sync-stripe"
---

Sync a product to Stripe. Creates or updates the corresponding Stripe product.

## Authentication

<Note>
This endpoint requires **secret key** authentication (`sk_live_...`).
</Note>

## Path parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `id` | `string` | Product ID. |

## Response

```json
{
  "stripe_product_id": "prod_abc123def456"
}
```

## Behavior

- If the product doesn't exist in Stripe, it creates a new Stripe product
- If the product already exists in Stripe, it updates the existing product
- The `stripe_product_id` is stored in the database for future reference

## Errors

| Code | Status | Description |
|------|--------|-------------|
| `K4001` | 404 | Product not found. |
| `K3002` | 403 | Insufficient permissions. |
| `K7001` | 502 | Stripe API error. |
| `K7002` | 400 | Stripe not connected. |
