---
title: "Get user"
api: "GET /v1/users/:id"
---

Retrieve a single user by their ID.

## Authentication

<Note>
This endpoint requires **secret key** authentication (`sk_live_...`).
</Note>

## Path parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `id` | `string` | User ID (e.g., `usr_01234567890abcdef`). |

## Response

```json
{
  "id": "usr_01234567890abcdef",
  "email": "user@example.com",
  "name": "John Doe",
  "role": "member",
  "status": "active",
  "email_verified": true,
  "created_at": "2025-01-01T00:00:00.000Z",
  "updated_at": "2025-01-01T00:00:00.000Z"
}
```

## Errors

| Code | Status | Description |
|------|--------|-------------|
| `K4002` | 404 | User not found. |
| `K3002` | 403 | Insufficient permissions. |
