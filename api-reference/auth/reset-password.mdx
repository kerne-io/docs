---
title: "Reset Password"
api: "POST /v1/auth/password/confirm"
---

Reset a user's password using a reset token.

## Authentication

<Note>
This is a **public** endpoint. The token must be valid and not expired.
</Note>

## Request body

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `token` | `string` | Yes | Password reset token from the email. |
| `password` | `string` | Yes | New password (min 8 characters). |

## Example request

```json
{
  "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
  "password": "newSecurePassword123"
}
```

## Response

```json
{
  "success": true,
  "message": "Password reset successfully"
}
```

## Errors

| Code | Status | Description |
|------|--------|-------------|
| `K1006` | 401 | Token expired. |
| `K1007` | 401 | Invalid token. |
| `K2002` | 400 | Password too weak. |
| `K2001` | 400 | Token and password are required. |
