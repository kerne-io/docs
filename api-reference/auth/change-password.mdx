---
title: "Change Password"
api: "POST /v1/auth/password/change"
description: "Change password for the currently authenticated user"
---

## Request Body

<ParamField body="currentPassword" type="string" required>
  The user's current password
</ParamField>

<ParamField body="newPassword" type="string" required>
  The new password to set
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  Whether the password was changed successfully.
</ResponseField>

<ResponseField name="message" type="string">
  A confirmation message.
</ResponseField>

## Errors

| Code | Status | Description |
|------|--------|-------------|
| `K1007` | 401 | **Token not provided**: No active session was found. |
| `K1005` | 401 | **Invalid credentials**: The current password is incorrect. |
| `K2001` | 400 | **Validation error**: The new password does not meet requirements. |
