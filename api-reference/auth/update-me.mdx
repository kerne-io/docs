---
title: "Update Me"
api: "PATCH /v1/auth/me"
description: "Update current authenticated user profile"
---

## Request Body

<ParamField body="first_name" type="string">
  User first name
</ParamField>

<ParamField body="last_name" type="string">
  User last name
</ParamField>

<ParamField body="avatar" type="string">
  User avatar URL
</ParamField>

## Response

<ResponseField name="user" type="User">
  The updated profile of the currently authenticated user.
</ResponseField>
