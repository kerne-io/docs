---
title: "Create product"
api: "POST /v1/billing/products"
---

Create a new product.

## Authentication

<Note>
This endpoint requires **secret key** authentication (`sk_live_...`).
</Note>

## Request body

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `name` | `string` | Yes | Product name. |
| `description` | `string` | No | Product description. |
| `type` | `string` | No | Product type (`subscription`, `one_time`). Default: `subscription`. |
| `metadata` | `object` | No | Custom metadata. |

## Example request

```json
{
  "name": "Enterprise Plan",
  "description": "For large organizations",
  "type": "subscription"
}
```

## Response

```json
{
  "id": "prod_01234567890abcdef",
  "name": "Enterprise Plan",
  "description": "For large organizations",
  "type": "subscription",
  "is_active": true,
  "metadata": {},
  "created_at": "2025-01-01T00:00:00.000Z"
}
```

## Errors

| Code | Status | Description |
|------|--------|-------------|
| `K2001` | 400 | Invalid request payload. |
| `K3002` | 403 | Insufficient permissions. |
