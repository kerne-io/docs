---
title: "List packs"
api: "GET /v1/packs"
---

List all packs (versioned entitlement snapshots) for a plan.

## Authentication

<Note>
This endpoint requires **secret key** authentication (`sk_live_...`).
</Note>

## Path parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `planId` | `string` | Plan ID. |

## Response

```json
[
  {
    "id": "pack_01234567890abcdef",
    "plan_id": "plan_pro_monthly",
    "version": 2,
    "status": "ACTIVE",
    "created_at": "2025-01-15T00:00:00.000Z",
    "entitlements": [
      {
        "id": "ent_abc123",
        "feature_id": "feat_api_calls",
        "feature_key": "api_calls",
        "feature_name": "API Calls",
        "feature_type": "metered",
        "limit": 10000,
        "value": null
      }
    ]
  },
  {
    "id": "pack_00123456789abcdef",
    "plan_id": "plan_pro_monthly",
    "version": 1,
    "status": "ARCHIVED",
    "created_at": "2025-01-01T00:00:00.000Z",
    "entitlements": [...]
  }
]
```

## Pack status

| Status | Description |
|--------|-------------|
| `DRAFT` | Pack is being configured, not yet usable. |
| `ACTIVE` | Current pack for new subscriptions. |
| `ARCHIVED` | Old pack, kept for grandfathered subscribers. |

## Errors

| Code | Status | Description |
|------|--------|-------------|
| `K4001` | 404 | Plan not found. |
| `K3002` | 403 | Insufficient permissions. |
