```
---
title: "List prices"
api: "GET /v1/billing/prices"
---

List all prices for a plan.

## Authentication

<Note>
This endpoint requires **secret key** authentication (`sk_live_...`).
</Note>

## Path parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `planId` | `string` | Plan ID. |

## Errors

| Code | Status | Description |
|------|--------|-------------|
| `K4001` | 404 | Plan not found. |
| `K3002` | 403 | Insufficient permissions. |
