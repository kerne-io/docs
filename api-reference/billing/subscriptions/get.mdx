---
title: "Get subscription"
api: "GET /v1/admin/billing/subscriptions/{id}"
---

Retrieve a subscription by ID.

## Authentication

<Note>
This endpoint requires **secret key** authentication (`sk_live_...`).
</Note>

## Path parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `id` | `string` | Subscription ID. |

## Response

```json
{
  "id": "sub_01234567890abcdef",
  "account_id": "acc_01234567890abcdef",
  "plan_id": "plan_pro_monthly",
  "plan": {
    "id": "plan_pro_monthly",
    "name": "Pro Monthly"
  },
  "status": "active",
  "current_period_start": "2025-01-01T00:00:00.000Z",
  "current_period_end": "2025-02-01T00:00:00.000Z",
  "stripe_subscription_id": "sub_abc123",
  "created_at": "2025-01-01T00:00:00.000Z"
}
```

## Errors

| Code | Status | Description |
|------|--------|-------------|
| `K4002` | 404 | Subscription not found. |
| `K3002` | 403 | Insufficient permissions. |
