---
title: "Create subscription"
api: "POST /v1/admin/billing/subscriptions"
---

Create a subscription for an account.

## Authentication

<Note>
This endpoint requires **secret key** authentication (`sk_live_...`).
</Note>

## Request body

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `account_id` | `string` | Yes | Account ID to subscribe. |
| `plan_id` | `string` | Yes | Plan ID to subscribe to. |
| `status` | `string` | No | Initial status (default: `active`). |

## Example request

```json
{
  "account_id": "acc_01234567890abcdef",
  "plan_id": "plan_pro_monthly"
}
```

## Response

```json
{
  "id": "sub_01234567890abcdef",
  "account_id": "acc_01234567890abcdef",
  "plan_id": "plan_pro_monthly",
  "status": "active",
  "created_at": "2025-01-01T00:00:00.000Z"
}
```

## Errors

| Code | Status | Description |
|------|--------|-------------|
| `K4001` | 404 | Account or plan not found. |
| `K2001` | 400 | Invalid request payload. |
| `K3002` | 403 | Insufficient permissions. |
| `K5005` | 409 | Account already has an active subscription. |
