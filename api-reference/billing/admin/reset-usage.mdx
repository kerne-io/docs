---
title: "Reset usage"
api: "POST /v1/admin/billing/usage/reset"
description: "Reset usage for a specific feature and scope"
---

## Authentication

<Note>
This endpoint requires **secret key** authentication (`sk_live_...`).
</Note>

## Request body

<ParamField body="scopeId" type="string" required>
  The ID of the user or project
</ParamField>

<ParamField body="featureKey" type="string" required>
  The key of the feature to reset
</ParamField>

## Response

Returns `204 No Content` on success.

## Errors

| Code | Status | Description |
|------|--------|-------------|
| `K1007` | 401 | **Unauthorized**: Invalid or missing secret key. |
| `K4002` | 404 | **Tenant not found**: The specified tenant slug is invalid. |
| `K3001` | 404 | **Feature not found**: The specified feature key does not exist. |
