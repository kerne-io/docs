---
title: "List overrides"
api: "GET /v1/admin/billing/overrides"
description: "List all entitlement overrides for a specific scope (user or project)"
---

## Query parameters

<ParamField query="scope_id" type="string" required>
  The ID of the user or project to list overrides for
</ParamField>

## Response

<ResponseField name="overrides" type="Override[]">
  A list of entitlement overrides for the specified scope.
</ResponseField>

### Override

<ResponseField name="id" type="string">
  The override ID
</ResponseField>

<ResponseField name="feature_id" type="string">
  The ID of the feature
</ResponseField>

<ResponseField name="limit" type="number">
  The custom limit (-1 for unlimited)
</ResponseField>

<ResponseField name="value" type="string">
  The custom value (for static features)
</ResponseField>

## Errors

| Code | Status | Description |
|------|--------|-------------|
| `K1007` | 401 | **Unauthorized**: Invalid or missing secret key. |
| `K4002` | 404 | **Tenant not found**: The specified tenant slug is invalid. |
