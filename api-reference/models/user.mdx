---
title: "User"
description: "The User object"
---

The **User** object represents a unique identity within a specific tenant.

<ResponseField name="id" type="string" required>
  The unique identifier for the user (UUID v7).
</ResponseField>

<ResponseField name="email" type="string" required>
  The user's email address. Unique per tenant.
</ResponseField>

<ResponseField name="first_name" type="string">
  The user's first name.
</ResponseField>

<ResponseField name="last_name" type="string">
  The user's last name.
</ResponseField>

<ResponseField name="is_active" type="boolean" required>
  Whether the user is allowed to log in.
</ResponseField>

<ResponseField name="global_role" type="string" required>
  The user's role in the system: `CLIENT` or `ADMIN`.
</ResponseField>

<ResponseField name="created_at" type="string" required>
  ISO 8601 timestamp of creation.
</ResponseField>

<ResponseField name="last_login_at" type="string">
  ISO 8601 timestamp of the last successful login.
</ResponseField>

```json Example
{
  "id": "01H8...",
  "email": "alice@company.com",
  "first_name": "Alice",
  "last_name": "Doe",
  "is_active": true,
  "global_role": "ADMIN",
  "created_at": "2023-01-01T12:00:00Z",
  "last_login_at": "2023-01-02T08:30:00Z"
}
```
